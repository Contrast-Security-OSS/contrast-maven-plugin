contrast:scan

Full name:

com.contrastsecurity:contrast-maven-plugin:2.13-SNAPSHOT:scan

Description:

Analyzes the Maven project's artifact with Contrast Scan to provide security insights

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 2.13.
  • Binds by default to the lifecycle phase: integration-test.
  • Requires that Maven runs in online mode.

Required Parameters

Name Type Since Description
<apiKey> String 2.13 API Key for communicating with Contrast. Find your personal keys
User property is: apiKey.
<organizationId> String 2.13 Unique ID for the Contrast Organization to which the plugin reports results. Find your Organization ID
Alias is: orgUuid.
<projectId> String 2.13 Contrast Scan project unique ID to which the plugin runs new Scans. This will be replaced imminently with a project name
<serviceKey> String 2.13 Service Key for communicating with Contrast. Find your personal keys
User property is: serviceKey.
<userName> String 2.13 User name for communicating with Contrast. Agent users lack permissions required by this plugin. Find your personal keys
Alias is: username.
